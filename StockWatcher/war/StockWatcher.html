Awards Watcher

María Jesús Platero Sánchez & Francisco Valladares Suárez

Requirements
1. There must be a data set which is stored at the server side (in a separate file or literally).
2. The user shall be able to add(download) data items from the data set to the table.
3. The user shall be able to remove data items from the table.
4. There shall be a coloring theme for the displayed data items.
5. At least One data property must change during the time by means of randomization.
6. The data items displayed in the table shall be updated regularly.
7. The user shall be able to view the time of the last update.
8. The user shall be able to add(upload) new data items to the remote data set.
9. The user shall be able to drag data items from the table and drop them into some target area.
the target area shall display information about the dropped data item.
